Latin [ edit ]

Etymology [ edit ]

From potēns +‎ -ia .

Pronunciation [ edit ]

( Classical ) IPA ( key ) : /poˈten.ti.a/ , [pɔˈt̪ɛn̪.t̪i.a] ( Ecclesiastical ) IPA ( key ) : /poˈten.t͡si.a/ , [pɔˈt̪ɛn.t͡si.a]

Noun [ edit ]

potentia f ( genitive potentiae ); first declension

force , power , might ability , capacity political power, authority , influence , sway crutch , walking aid (Middle Latin only)

Declension [ edit ]

First-declension noun.
